프로젝트: 맘스터치 사이트 리뉴얼

프로젝트 소개: 기존 사이트의 불편한 가독성과 디자인을 수정하고, 이용자들이 더 편리한 웹페이지 사용을 할 수 있도록 리뉴얼하는 것을 목표로 삼음. 수정해야 할 사항으로는, 좌측 상단의 버거 버튼을 클릭하면 나타나는 메뉴창의 부자연스러운 디자인, 제품의 이름보다 설명을 더 강조한 디자인, 홈으로 돌아가면 팝업창이 계속해서 뜨는 것 등의 문제를 발견함.

참고 사이트: https://www.momstouch.co.kr/

참여도: 100% 개인 작업

사용 툴: HTML, CSS, jQuery, Figma

1. 원하는 제품이나 질문을 바로 검색할 수 있는 서치 바

2. 제품 홍보 이미지 슬라이드, 일정 시간이 지나면 자동으로 슬라이드된다.

3. 제품 매뉴를 바로 확인할 수 있는 카테고리와 인기 메뉴 순으로 나타나는 제품 이미지들, 양 옆의 화살표 버튼을 누르면 다음 이미지로 슬라이드된다.

4. 현재 위치 근처의 매장을 보여주는 미니맵

5. 클릭하면 배달 어플 앱으로 연결되는 링크가 뜨는 버튼

맘스터치의 컨셉 컬러 세 가지를 컬러 가이드로 잡고 작업했습니다.

메인 페이지의 글꼴은 이목을 끌고, 가독성이 좋은 SB 어그로체를, 푸터의 글꼴은 단정하고 깔끔한 느낌의 Noto Sans KR체를 사용하였습니다.
