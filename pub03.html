프로젝트: 호텔 사이트 메인 페이지 제작

프로젝트 소개: 호텔 사이트 하나를 참고로 직접 PC 버전 메인 페이지와 모바일 버전 메인 페이지를 디자인함. 메인 테마를 해변가의 휴양지, 보랏빛 컬러로 정하고 작업.

참고 사이트: https://www.commodore.co.kr/html/main/

참여도: 100% 개인 작업

사용 툴: HTML, CSS, jQuery, Figma

- 로고를 클릭하면 메인 화면으로 돌아감

- 상단 메뉴를 클릭하면 하위 메뉴가 나타남

- 체크인, 체크아웃, 인원수, 객실 칸에 직접 입력하거나, 클릭하면 나타나는 선택창으로 값을 입력하고 검색할 수 있다.

- 가장 인기 있는 유형의 룸 사진을 띄우고, 하단에 예약 버튼을 배치한다. 예약 버튼을 누르면 룸 예약 페이지로 이동할 수 있다.

- 호텔 근처의 유명 관광지를 배경으로 한 투어가이드 배너를 띄운다. 누르면 해당 안내 페이지로 이동한다.
