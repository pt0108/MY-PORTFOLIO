프로젝트: 작가 소개 사이트 메인 페이지 제작

프로젝트 소개: 소개하고 싶은 작가 한 명을 정해서 모바일 와이어 프레임과 PC용 메인 페이지, 서브 페이지의 와이어 프레임을 제작함. 소개할 인물로는 어린이 동화책 작가로 유명한 앤서니 브라운을 채택함. PC용 메인 페이지는 디자인과 코딩도 하였음.

참여도: 100% 개인 작업

사용 툴: HTML, CSS, jQuery, Figma

- 첫번째 이미지는 PC용 메인 페이지의 와이어 프레임. 전시 일정과 추천 도서, 공모전 등을 간편하게 확인할 수 있도록 설계함.

- 두번째 이미지는 PC용 서브 페이지 와이어 프레임. ABOUT 메뉴의 상세 페이지를 설계함. 작가의 프로필과, 대표 시리즈를 연도별로 정리하였음.

- 세번째 이미지는 모바일용 와이어 프레임. 버거 버튼을 누르면 나타나는 메뉴창의 모습도 설계함.
